December 27, 2022 in Hugo, post

Introduction

I thought to build a personal website several times, but failed to do it. At one weekend of December, 2022 I decided to complete this website project before the new year 2023. I started by searching “create a personal website” in youtube. The algorithm recommends videos with Hugo. My first thought ðŸ¤”: since when Hugo Boss started to build websites instead of selling expensive clothes ðŸ˜„. Some videos promise to do it with Hugo within 10 minutes. That’s great. I was excited about it.

The main reasons I want to have a personal website are:

  • writing blogs: I continue learning new skills and technologies. I want to summarize and document them down by writing blogs.
  • digital portfolie: showcase my past projects and share knowledge.

Hugo is a static site generator which allows website authors to create blogs or pages in Markdown. So it’s suitable for me. You can host it freely in Github pages. What’s more, there are hundreds of Hugo themes ready to use for various purpose. Sounds cool? And guess what? I spent 10 minutes to watch that youtube turorial but the rest of weekend to figure it what’s wrong with one error by deploying. I will share my guide so that you don’t have to go through all the pain I went throught.

Prerequisites

In order to contiue and see if it’s suitable for you, you need to fill the following prerequisites.

  • Installation of Hugo
    Refer to the official documentation Installation
  • Basic knowledge of Markdown or HTML
    Both HTML and Markdown are supported content formats.
  • One IDE, e.g. VS code
  • A Github account
    Ahttps://pages.github.com/
  • Knowledge of Git
    Git is required to install a theme as a Git submodule, access commit information from a local repository and host your site.

Choose a Hugo Theme

You can find one suitable Hugo theme hier. Either for a portfolio, or company, there are plenty of template themes to choose. I’ve chosen hugo tranquilpeak theme.
